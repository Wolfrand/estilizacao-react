## Estilização no React

No React, podemos aplicar estilos de diversas maneiras. As principais são:

- CSS tradicional (externo) – arquivos .css comuns;
- CSS Modules – estilos isolados para cada componente;
- Styled Components – CSS-in-JS, criando componentes já estilizados;
- Bibliotecas de estilização – como TailwindCSS, SASS, entre outras.
